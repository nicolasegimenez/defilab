🏗️ Arquitectura del Sistema

DefiLab sigue una arquitectura modular basada en componentes reutilizables, separando claramente la estructura, el estilo y la funcionalidad.

📁 Estructura de Archivos

defilab/
├── styles/          # Hojas de estilo CSS
├── docs/            # Documentación técnica
├── components/      # Componentes UI
└── index.html       # Punto de entrada

Ejemplo de Uso: Gráfico Financiero

<!-- Incluir Chart.js -->
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

<!-- Contenedor del gráfico -->
<canvas id="miGrafico"></canvas>

<!-- Configuración en JavaScript -->
<script>
  const ctx = document.getElementById('miGrafico');
  new Chart(ctx, {
    type: 'line',
    data: {
      labels: ['Ene', 'Feb', 'Mar', 'Abr'],
      datasets: [{
        label: 'Precio Token',
        data: [120, 150, 180, 220],
        borderColor: '#4f46e5'
      }]
    }
  });
</script>

🛠️ Guía de Instalación y Configuración

Sigue estos pasos para configurar DefiLab en tu entorno local.

Requisitos Previos

  • Navegador web moderno (Chrome, Firefox, Edge)
  • Conexión a internet (para CDN de dependencias)
  • Editor de código (VS Code recomendado)

Pasos de Instalación

  1. Clonar el repositorio: 
    git clone https://github.com/nicolasegimenez/defilab.git
cd defilab
  2. Abrir el archivo principal: 
    Abre index.html en tu navegador
  3. Personalizar estilos (opcional): 
    Edita los archivos en la carpeta styles/
  4. Desplegar en producción: 
    Sube los archivos a tu hosting o usa GitHub Pages

Estructura de Carpetas

defilab/
├── index.html          # Página principal
├── styles/
│   ├── main.css       # Estilos globales
│   └── components.css # Estilos de componentes
├── docs/
│   ├── diagrams/      # Imágenes de diagramas
│   └── index.html     # Esta documentación
└── README.md          # Documentación principal
